# Owner Guide

Nati Nest lets customers order from their table using a QR code while kitchen, server, and admin staff manage the full canteen workflow.

## What the System Does

- Customers scan a table QR code.
- Customers browse the menu and place orders.
- Kitchen receives new orders instantly.
- Server staff deliver ready orders and resolve customer requests.
- Customers request bills and pay by cash or UPI.
- Admin tracks menu, tables, staff, reports, settings, and catering leads.

## Daily Owner Checklist

- Confirm the system is online.
- Confirm kitchen and server tablets show live connection.
- Check unavailable menu items before service starts.
- Review payment settings.
- Review previous day revenue and unpaid/pending items.
- Confirm backup job status.

## Key Business Screens

| Screen | Use |
| --- | --- |
| Admin Dashboard | High-level operations summary |
| Admin Menu | Categories, menu items, pricing, availability, images |
| Admin Tables | Table QR setup and table status |
| Admin Staff | Staff accounts and active/inactive status |
| Admin Reports | Revenue, orders, feedback, tips, table reports |
| Admin Settings | Business profile, UPI, logo |
| Catering | Catering enquiries and status pipeline |

## Launch Readiness

Do not launch until:

- Production secrets are configured.
- Seed passwords are changed.
- QR codes are printed and tested.
- Staff have completed a full test order.
- Payment verification is tested.
- Backup and restore are confirmed.
