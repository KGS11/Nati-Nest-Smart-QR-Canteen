# Administrator Guide

Admins use the `/admin` dashboard and child pages to manage the canteen.

## Menu Management

Use Admin -> Menu:

- Create, edit, and delete categories.
- Create, edit, and delete menu items.
- Upload item images.
- Toggle item availability.
- Mark popular items.

Operational rule: prices must be maintained in the backend database. Customer order totals are calculated from database prices.

## Table Management

Use Admin -> Tables:

- Create physical tables.
- Update table labels/numbers.
- Regenerate QR identifiers.
- Track table status.
- Delete unused tables.

Print and test each table QR before service.

## Staff Management

Use Admin -> Staff:

- Create admin, kitchen, and server users.
- Update staff details.
- Activate/deactivate accounts.
- Rotate passwords when staff leave.

## Settings

Use Admin -> Settings:

- Business name.
- Address.
- Phone.
- Tax rate.
- UPI ID.
- Static UPI QR image.
- Logo image.

## Reports

Use Admin -> Reports:

- Dashboard summary.
- Revenue report.
- Orders report.
- Popular items.
- Table-wise report.
- Feedback analytics.
- Tips report.
- Export reports where supported.

## Daily Menu

Use Admin -> Daily Menu:

- View today's available menu.
- Remove items with a reason.
- Restore removed items.
- Copy yesterday's menu.
- Review history.

## Catering

Use Admin -> Catering:

- Review new enquiries.
- Search by name or phone.
- Move statuses: `NEW`, `CONTACTED`, `QUOTED`, `WON`, `LOST`.
- Export leads/enquiries.
